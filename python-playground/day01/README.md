# Python 第一天：基础语法

## 今日目标

今天只掌握 5 件事：

1. 会运行 Python 文件。
2. 会使用 `print()` 输出内容。
3. 会定义变量。
4. 会写 `if / elif / else` 条件判断。
5. 会写 `for` 循环。

## 学习顺序

1. 先运行 `hello.py`。
2. 再打开 `practice.py`。
3. 按注释完成练习。
4. 每改一次代码就运行一次。

## 前端类比

| 前端 JavaScript | Python |
| --- | --- |
| `console.log()` | `print()` |
| `let name = 'Ace'` | `name = 'Ace'` |
| `{}` 代码块 | 缩进代码块 |
| `true` / `false` | `True` / `False` |
| `null` / `undefined` | `None` |
| `for (let i = 0; i < 5; i++)` | `for i in range(5)` |

## 今日练习

完成 `practice.py` 中的 5 个练习。

## 注意事项

- Python 靠缩进表示代码块，不能随便多空格或少空格。
- 布尔值首字母大写：`True`、`False`。
- 空值是 `None`，不是 `null` 或 `undefined`。
- 字符串推荐使用 f-string：`f"你好 {name}"`。
